My First Website

Purpose

This is the first website I have created for IS601. The purpose of this website is to familiarize myself with using git.
Another objective of it is to help me grow accustomed to using Linux and Vi commands. Below are some of the commands that I used during the creation process.

Linux Commands

  • pwd = Show Current Directory
  • ls = Directory Listing
  • cd = Change Directory
  • mkdir = Make Directory

Vi Commands

  • vi FILENAME = Opens Vi Editor
  • i = Enter Insert Mode
  • esc key = Exit Insert Mode
  • shift + colon = Command Prompt for Vi
  • W = Command to Write
  • Q = Command to Quit
  • WQ = Command to Write and Quit

Git Commands

  • git init = Creates the Repository
  • git status = Shows modified files ready for a commit
  • git add FILENAME = Adds the file to git in preparation for a commit
  • git commit -m "MESSAGE" = Does the commit to save changes
  • git checkout -b BRANCH = Makes a new branch and checks it out
  • git checkout BRANCH = Check out a specified branch
  • git merge BRANCH = Merges the specified branch into the branch that you are on
